猫咪趣事

从那时起，正如一切竹马都打不过天降，这只与泡脚凤爪一并出现的怂猫，渐渐与官方名称泡椒解绑，被人们习惯性地称为大哥。
雄风不在的香波终于宣告彻底康复，回到了他的故土。
薏米，传说之中的燕南女明星，安定万人迷。她，行走在燕园的花丛中却不为片叶停留；她，异色的瞳孔中流露出的是对红尘情事的淡薄。
猫协与电教的故事，大约也就这样结束了吧。
脏鼻儿是出没于南北阁附近的一只橘猫，因为鼻尖上一直有一抹黑，被我们叫做“脏鼻儿”。
橘色的二胡就呆在枝叶之门前面、杂草野花掩映之间，和它的毛色一样温暖的午后阳光落下来。装在镜头里，这是一幅俨然童话的景象。
